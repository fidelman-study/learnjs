Конструктором становится любая функция, вызванная через new.

Чтобы выделить конструкторы их называют с большой буквы

если у функции-конструктора нет аргументов,, можно вызывать без скобок

функцию и называют конструктором – она предназначена для «конструирования» объектов.

Для создания одного объекта используют конструкцию new function() { … }

вызов return с объектом вернёт объект, а с чем угодно, кроме объекта – возвратит, как обычно, this.
