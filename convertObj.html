Строковое состояние [object Object]

Метод toString - возвращает то, как будет выглядеть объект в строковом состоянии

Метод valueOf - возвращает то, как будет выглядеть объект в числовом состоянии, Метод valueOf обязан возвращать примитивное значение, иначе его результат будет проигнорирован и вызовется toString

Но оператор + для Date использует именно toString (хотя должен бы valueOf)

если { ... } находится вне выражений, то это не объект, а блок.
